## Installation 
1. Create DB Blog to your Local Database
2. Setting configuration database in .Env
3. run 'composer install && composer update'
4. run 'php artisan migrate' to migrate all table
5. run 'composer dump-autoload' to create autoload
6. run 'php artisan db:seed' to add seed data in database
7. run 'php artisan storage:link' to add storage link
8. run 'php artisan serve'
